Description
-----------

An OBJ file of a triangulated mesh with texture coordinates and a bitmap
texture for a spotted animal homeomorphic to a sphere.

The original data comes with Catmull-Clark control mesh, quadrangulation,
triangulation, vector texture, and bitmap texture.

Source: http://www.cs.cmu.edu/~kmcrane/Projects/ModelRepository/
